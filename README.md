Business Contact Page

A simple, mobile-friendly contact page that allows visitors to choose between WhatsApp and SMS to get in touch.

About

This page provides two easy ways for customers to contact the business:
- **WhatsApp** - Opens a WhatsApp chat
- **SMS** - Opens the default messaging app

 Features

- Clean, modern design
- Mobile-responsive
- Works on all devices and browsers
- Easy to update contact information

How to Use

Simply scan the QR code or visit the website URL, then choose your preferred contact method.

Updating the Phone Number

To change the phone number:

1. Open `index.html`
2. Find all instances of the number
3. Replace with your new number (include country code without + or spaces)
4. Save the file

QR Code

Generate a QR code pointing to your GitHub Pages URL:
`https://YOUR-USERNAME.github.io/contact/`
